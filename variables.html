Variables

Variables, in computer programming, hold data that can be used, manipulated, or changed. Variables can be:

  • Integers
    • 5
    • -6
  • Floating Point Numbers
    • 6.34
    • -12.52
  • Strings
    • "Hello."
  • Characters
    • 'g'
    • 'p'
  • Boolean Values
    • True
    • False

These are just examples of values you can assign. There's quite a lot more you can do with these. Here's an example writtenin C++: 

int integerVariable = 100;
float floatingPointVariable = 123.62;
std::string stringVariable = "Hello, World!"; // Strings use double quotes
char characterVariable = 'k'; // Characters use single quotes
bool booleanVariable = false;

By the way, those // symbols start a comment, which can help you understand code. You can also change variables through code. Here's how (in C++):

integerVariable = 120;
integerVariable += 30; // integerVariable now has the value of '150'

// You can do similar things with floats, too

stringVariable = "This is a new string";
// You can also concatenate strings 
stringVariable += ", and now it is newer"; // stringVariable now has the value of "This is a new string, and now it is newer"

characterVariable = 'd';
// You cannot concatenate char's (or booleans), as far as I know, because they can only hold one character

booleanVariable = true;

Maybe you've kept track of a score, someone's name, or something else important (or even unimportant), but you don't know how to use it. Well, all you got to do is write the name. Here's an example in C++:

#include <iostream> /*Allows printing to the console*/

int main()
{
    std::string name = "John";

    std::cout << "Hello, " << name /*The variable defined earlier*/ << std::endl; // Prints "Hello, John" to the console
    return 0; // Ends the program
}
